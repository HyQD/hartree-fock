Ground state calculations¶

This library provides three Hartree-Fock solvers, they are:

  • Hartree-Fock solver with general spin-orbitals, HartreeFock.

  • The restricted Hartree-Fock method, RHF.

  • The unrestricted Hartree-Fock method, UHF.

All classes re-use the self-consistent field procedure defined in the HartreeFock-class, but implement their own methods where needed, e.g., the construction of the Fock matrix.

class hartree_fock.hf.HartreeFock(system, mixer=<class 'hartree_fock.mix.EmptyMixer'>, verbose=False, np=None)¶

Class implementing a Hartree-Fock solver using general spin-orbitals.

Parameters

  • system (QuantumSystem) – System containing matrix elements needed by the solver.

  • mixer (Mixer) – Convergence mixer for the self-consistent field procedure. Default is EmptyMixer which does no mixing. This is enough in many cases.

  • verbose (bool) – Variable to toggle the output of convergence info and converged energy. Default is False, i.e., quiet-mode.

  • np (module) – Matrix library, default is None which imports numpy.

change_basis()¶

Function changing the system basis to the Hartree-Fock basis.

compute_density_residual(prev_density_matrix, density_matrix)¶

Function computing the residual of the density matrices for the convergence test of the self consistent field procedure. That is,

\[\rvert\rvert \Delta \mathbf{D} \lvert\lvert_{F} = \rvert\rvert \mathbf{D}^{(i + 1)} - \mathbf{D}^{(i)} \lvert\lvert_{F}\]

where \(\mathbf{D}^{(i)}\) is the density matrix at iteration \(i\) of the self consistent procedure.

Note that this function works for both single density matrices, and a list of density matrices. This lets the UHF-class reuse this procedure.

Parameters

  • prev_density_matrix (np.ndarray, list) – The density matrices at the previous timestep \(i\).

  • density_matrix (np.ndarray, list) – Density matrices at the current timestep \(i + 1\).

Returns

The residuals from given density matrices.

Return type

list

compute_energy()¶

Compute general Hartree-Fock energy, given by

\[E = D_{\beta \alpha} h_{\alpha \beta} + \frac{1}{2} D_{\gamma\alpha} D_{\delta\beta} u^{\alpha\beta}_{\gamma\delta},\]

where \(D_{\beta\alpha}\) are the matrix elements of the density matrix, \(h_{\alpha\beta}\) the one-body Hamiltonian matrix elements of the atomic orbitals, and \(u^{\alpha\beta}_{\gamma\delta}\) the antisymmetric two-body Hamiltonian matrix elements of the atomic orbitals.

Returns

The Hartree-Fock energy for general spin-orbitals.

Return type

complex

compute_ground_state(max_iterations=100, tol=0.0001, change_system_basis=False, **mixer_kwargs)¶

Compute the Hartree-Fock ground state using self consistent field iterations.

Parameters

  • max_iterations (int) – The maximum number of self consistent iterations. Default is 100.

  • tol (float) – The convergence tolerance for the energy and density residuals. Default is 1e-4.

  • change_system_basis (bool) – Toggle whether or not to change to the Hartree-Fock basis after the self consistent iterations have converged.

  • **mixer_kwargs (dict) – Keyword arguments to the Fock matrix mixer class.

compute_initial_guess()¶

Compute initial guess for the self-consistent field procedure using the one-body Hamiltonian as the initial Fock matrix and solving the Roothan-Hall equations. That is, the function solves the generalized eigenvalue equation

\[\boldsymbol{h}\boldsymbol{C} = \boldsymbol{S}\boldsymbol{C}\boldsymbol{\epsilon},\]

where \(\boldsymbol{h}\) is the one-body Hamiltonian, \(\boldsymbol{C}\) the coefficient matrix as the eigenvectors, \(\boldsymbol{S}\) the overlap matrix from the system, and \(\boldsymbol{\epsilon} = \text{diag}(\epsilon_1, \dots)\) the diagonal matrix with the eigenvalues from the one-body Hamiltonian. The function proceeds by building and storing the density matrix \(\boldsymbol{D}\) from the coefficient matrix, and the initial Fock matrix \(\boldsymbol{F}\).

Note that this function is called internally by the compute_ground_state-function.

compute_one_body_density_matrix()¶

Compute the one-body density matrix from the occupied coefficient matrix. This is done by

\[\rho^{q}_{p} = \langle\Phi\rvert \hat{c}^{\dagger}_{p} \hat{c}_{q} \lvert\Phi\rangle = \delta_{p \in o} C^{*}_{\alpha p} s_{\alpha \beta} C_{\beta q},\]

where \(\delta_{p \in o}\) denotes occupied indices \(o = {1, \dots, N}\), \(s_{\alpha\beta}\) are the overlap integrals of the atomic orbitals, and \(C_{\alpha p}\) the coefficient matrix. In case of converged self consistent iterations, the one-body density matrix should yield the identity along the occupied indices and zero elsewhere.

Returns

The one-body density matrix \(\rho^{q}_{p}\).

Return type

np.ndarray

compute_particle_density()¶

Function computing the particle density \(\rho(x)\) defined by

\[\rho(x) = \phi^{*}_{q}(x) \rho^{q}_{p} \phi_{p}(x),\]

where \(\phi_p(x)\) are the single-particle functions, \(\rho^{q}_{p}\) the one-body density matrix, and \(x\) some coordinate space. Note the use of the Einstein summation convention in the above expression.

Returns

Particle density on the same grid as the single-particle functions.

Return type

np.ndarray
